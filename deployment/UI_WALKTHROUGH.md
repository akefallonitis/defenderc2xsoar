## Tips for Users

### Best Practices
1. **Select devices first** before choosing action type
2. **Use multi-select** for bulk operations
3. **Monitor Action Manager** for real-time status
4. **Review device list** periodically to ensure accuracy
5. **Use filters** in tables to find specific devices/actions

### Shortcuts
- **Ctrl+Click** - Select multiple non-consecutive devices
- **Shift+Click** - Select range of devices
- **Ctrl+A** - Select all devices (when dropdown focused)

### Troubleshooting
- **Empty dropdown?** Check FunctionAppName and TenantId
- **Slow loading?** Check network connection and Function App status
- **Wrong devices?** Verify workspace and subscription selection
